Ambient Gradients

Very low-opacity radial gradients in the accent colour create atmospheric depth without visual noise. The intent is subliminal — the viewer should feel warmth without identifying a gradient.

Hero / Page Ambient
radial-gradient(ellipse 65% 50% at 38% 62%,
rgba(190, 252, 66, 0.028) 0%, transparent 70%)

Usage: hero sections, inner page heroes.
Position shifts per page for variety.

Dependencies

The site has zero JavaScript dependencies and zero CSS frameworks. The only external resources are Google Fonts (CDN) and the logo PNG.

ResourceTypeSourceNotes
DM Serif Display Font fonts.googleapis.com Weights: 400, 400 italic. ~22KB WOFF2.
Chivo Mono Font fonts.googleapis.com Weights: 300, 400. ~18KB WOFF2.
DM Sans Font fonts.googleapis.com Weights: 300, 400, 500 (variable). ~24KB WOFF2.
Logo_Hi_Res.png Image Local (assets/) Must sit in assets/ folder relative to HTML files. Used with mix-blend-mode:screen.
Google Fonts preconnect Network hint fonts.googleapis.com + fonts.gstatic.com Add rel="preconnect" link tags before the font href for performance.

Browser support — All modern browsers (Chrome 88+, Firefox 85+, Safari 14+, Edge 88+).
Features used: CSS custom properties, CSS Grid, IntersectionObserver, backdrop-filter, mix-blend-mode, clamp().

Performance notes
All JS is inline and deferred by placement at end of <body>. No external scripts.
IntersectionObserver uses threshold 0.1 for reveal triggers (0.3 for number bars).
Fit-text uses binary search on document.fonts.ready — requires fonts loaded before firing.

Offline / self-hosted fonts
For intranet or air-gapped deployments, download fonts via google-webfonts-helper.herokuapp.com,
place WOFF2 files in assets/fonts/, and replace the Google Fonts <link> with local @font-face declarations.
